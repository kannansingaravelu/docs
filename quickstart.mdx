## Installation

<AccordionGroup>
  <Accordion title="Create a virtual environment" icon="rectangle-terminal">
    <CodeGroup>

    ```shellscript Mac / Linux highlight={1}
    mkdir pythonlab && cd pythonlab
    
    uv venv .venv
    source .venv/bin/activate
    ```

    ```shellscript Windows (Cmd)
    mkdir pythonlab; cd pythonlab
    
    uv venv .venv
    .venv\Scripts\activate.bat
    ```

    ```shellscript Windows(Powershell)
    mkdir pythonlab; cd pythonlab
    
    uv venv .venv
    .venv\Scripts\Activate.ps1
    ```

    </CodeGroup>

    <Tip>
      Open the `Terminal` and create an `pythonlab`directory with a python virtual environment.
    </Tip>
  </Accordion>
</AccordionGroup>

## Import

<AccordionGroup>
  <Accordion title="Create a virtual environment" icon="rectangle-terminal">
    ```python
    from quantmod.models import OptionInputs, Black76
    
    b76 = Black76(OptionInputs(spot=100, strike=100, rate=0.05, ttm=1.0, volatility=0.20))
    
    print(b76.results())
    ```
  </Accordion>
</AccordionGroup>
